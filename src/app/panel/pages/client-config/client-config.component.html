Installation(manual):

2. Give it execution rights:
chmod +x cmoncli
3. Follow 'Systemd service' instructions
Systemd service:
1.
touch /usr/lib/systemd/system/cmoncli.service
2. Paste following content:
3.
cp cmoncli /usr/bin/cmoncli
4.
systemctl daemon-reload
5.
systemctl start cmoncli
6.
systemctl enable cmoncli
